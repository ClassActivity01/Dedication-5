2.2 Maintain Customer


Purpose of this function: To update a customer’s details to the system to be used when processing orders and generating financial documents, or to delete the customer from the system.


Navigation to add a new customer



The Maintain Customer screen is where you will first input the customer’s details for the system to use as search criteria. You will then select the customer you need to maintain from the displayed customers that the system found, and then provide the new customer’s details.



Screen Controls



The numbers indicated on the screens represent the following controls

  1. Search Criteria textbox: information about the customer is entered here to be used as a keyword for Proteus to use to find the customer you are searching for.
  2. Search by drop-down list: select the option which describes the information you typed in the Search Criteria textbox which Proteus will search through to find the customer.
  3. Search Method radio-edit box: select the option which defines how Proteus should search using the search criteria you provided.
  4. Search button: click this button to search for the customer
  5. Customer tab: on the click of the of this tab the required customer information will be displayed.
  6. Discounts tab: on the click of this tab the required customer discount information will be displayed
  7. Customer Name textbox: enter the customer’s company name here
  8. Account Name: enter the customer’s account name here
  9. VAT Number: enter the customer’s VAT number here
  10. Status drop-down list: select the customer’s status here
  11. Address textbox: enter the customer’s delivery street address here
  12. City textbox: enter the customer’s delivery address’s city name here
  13. Province drop-down list: select the customer’s delivery address’s province name here
  14. Zip code drop-down list: select the customer’s postal code here
  15. Add Contact button: click this button to add another contact person for the customer
  16. Remove contact button: select the contact and then click this button to remove the contact person for the customer
  17. Contact Name textbox: enter the contact person’s name here
  18. Contact Number textbox: enter the contact person’s contact number here
  19. Contact Email textbox: enter the contact person’s email address here
  20. Job Description textbox: enter the contact person’s job description here
  21. Update Customer button: click this button to save the updated customer details
  22. Remove Customer button: click this button to remove the all the customer’s details from the system



The numbers indicated on the screen represent the following controls:

  1. Customer tab: on the click of this tab the required customer information will be displayed
  2. Discount tab: on the click of this tab the required customer discount information will be displayed
  3. Contract Discount Rate numeric up-down: enter the contract discount rate offered to the customer here
  4. Settlement Discount Rate numeric up-down: enter the settlement discount rate offered to the customer here
  5. Add Discount to Part Type button: click this button to search for a part type to select and add a part discount rate for it
  6. Discount % textbox: enter the part discount rate here
  7. ‘Pompie Pump’ button: this button name depends on the part type listed for discount, click this button to remove the part type from the part discounts list


Process to follow:



Enter the customer’s company name,  VAT number, Account Number or customer number into the Search Criteria textbox for it to be used as search criteria.




Select an option from the Search By drop-down list which describes the information you typed in the Search Criteria textbox. The option you have selected is what Proteus will search through the database to find customers with details that match the information you provided in the Search Criteria textbox, i.e. if you select name then Proteus will search through all the customer’s names to find and display one/s that match the name you provided as search criteria. If you select the ‘All’ option, then Proteus will search through all the information stored about the customer for matching information and not just one specific detail about the customer like it would if you selected a different option.






Select an option in the Search Method radio-edit box which defines how Proteus should search using the search criteria you provided. Selecting ‘Exact’ means that any customer/s with details that match the search criteria you provided exactly. Selecting ‘Contains’ means that any customer/s with details that contain the information you provided will be found and displayed, i.e. if the customer’s details contain the search criteria you provided as a part within its whole detail then it will be selected, an example being if you provided “Prot” as search criteria and selected ‘Contains’ as the search method, then a customer “Proteus” would be found and displayed if it were an existing customer.




Click the Search button once you have provided information in the Search Criteria textbox and selected options in the Search By list-box and the Search Method radio-edit box. Where any matches have been found with the information you provided it will be displayed for you to view and select the exact customer you are searching for.



The Select a Customer message-box will be displayed once you have clicked the Search button and a list of all the customer’s that matched the search information that you provided (see above screen controls) will be shown in the list-box. Click on the customer that you want to maintain in the list-box and then click the Load Customer Details button to proceed to change that customer’s details or delete the customer. Click on the Close button if you want to close the message-box and change your provided search information or to discontinue the process of maintaining a customer’s details and select a different action to perform from the navigation bar.



After you selected the customer in the list-box and clicked the Load Customer Details button on the Select a Customer message-box, this Maintain Customer screen will display with the above controls containing all the details saved for the selected customer. All the details for this customer may be changed excluding it’s allocated Customer Number. Select on any of the textboxes and update the customer’s details.



Change the Status drop-down list to ‘In-Active’ if this customer no longer exists or if you no longer process continue to do business with the customer but have previously processed orders for this customer. Change the Status drop-down list to ‘Active’ if it was previously set to ‘In-Active’ and you now would like to be able to perform business together and process customer orders for this customer.




Click the Add Contact button to add another contact persons details for this customer. Once you have clicked this button the required form controls with appear for you to fill in the necessary details for the contact person.

Select a specific contact person and click the Remove Contact button to remove the selected contact person’s details as a contact person for this customer.



Click on the Discounts tab to change this discounts offered to this customer. Click on the relevant textboxes to update the details. Click on the ‘Pompie Pump’ button in the Parts with Discounts table to remove the part type from the list, this button will be named according to the part type it represents.



Click on the Add Discount to Part Type button to add a new part discount rate. An Add Part Type message-box will appear for you to search for the part type which you would like to provide a discount for this specific customer. This discount will only be used when this customer orders the selected part type.




Once you’ve clicked the Add Discount to Part Type button, the Add Part Type message-box will display. Input the part type’s search criteria in the textbox, and then select what the criteria should be filtered by in the drop-down list. Click the Filter button to then have the search results displayed in the listbox.



Click on one of the parts displayed in the listbox so that it is highlighted blue, and then click the Add Part Type to Discounts button to add the part’s details to the part discounts list and later set the discount.

If the part type you are looking for does not display in the listbox then click the Add New Part Type to System button to then open the screen to add a new part type and then return to this Add Customer screen. Once you have added all the part type’s details on the screen and then returned to this one, you will then need to repeat the search (see above paragraph on how to search for a part type) and can then continue to click on the part type and click the Add Part Type to Discounts button.

Once you are finished searching and adding the part types to provide discounts for, you can click the close button on the top right corner of the messagebox, this will then close Add Part Type message-box and allow you to set the part type’s discount rate in the relevant textbox in the table.

Close the message-box and then type in the discount rate for each part that you added in the Discount %  textbox.

Click the Customer tab to return to the customer’s details information once you are done setting the discounts.



Click on the Customer tab to return to it, this tab contain the customer’s details and the contact persons’ details. To save the newly updated details you will need to return to this tab.



Click the Update Customer button to save all the customer’s updated details. All textboxes must contain information for a customer to be successfully updated.



A confirmation message will display if the customer’s details have been successfully updated.



Click the Remove Customer button to delete the customer from the database. This action is permanent and may only be allowed if no orders have been processed for this customer.



Click OK if you want to confirm the deletion of this customer. Click CANCEL if you want to return to the Maintain Customer screen.



Once you’ve clicked the OK button to confirm the deletion of the customer this confirmation message-box will appear meaning the customer has been successfully deleted from the system. Click the OK button to dismiss the message-box.


System Error Notifications and Troubleshooting

**error message for if +100% discount is given


The system will highlight empty fields that require information to be provided, as well as fields which have the wrong information in. A customer may only be successfully maintained if all the required information has been provided and the information is in the correct format.



An error message similar to the one above may also be displayed at the bottom left of the screen near the Update Customer button, describing the type of error to explain to you why the customer cannot be added. Read the error message to understand what is wrong and click on the textboxes which have the wrong information causing the error, and correct the information. Once all the incorrect information in the textboxes has been corrected, click on the Update Customer button to then try again to successfully add the customer’s details. The ‘One or more fields are empty’ error will always be displayed over any other error, so you will first have to ensure that all the fields have been filled in and then only the invalid information errors will display.



When searching for a customer to maintain, if you click the Load Customer Details button on the Select a Customer message-box without having selected a customer from the list of customers displayed, this error will come up. Click the OK button to dismiss the error message and then click on a customer’s name from the list displayed so that it highlights blue (see image below), then click the Load Customer Details button to maintain the selected customer.




If you cannot find the customer you are searching for in the list displayed, then click the Close button and enter new information for Proteus to search for through the database of customers. Consider the fact that the customer may not exist on the system and that you might have to instead go to the Add Customer screen, otherwise try make the search a bit less specific for example only typing a portion of the customer’s name and the selecting ‘Contains’ as the Search Method, or select ‘All’ from the Search By drop-down list.




This error message displays when you try to delete the last contact person. A customer must have at least one contact person. Click the OK button to dismiss the notification.



This error message displays when you haven’t selected a part type from the Add Part Type message-box but clicked the Add Part Type to Discounts button. Click the OK button to dismiss the error message and then select a part type name from the list displayed so that it is highlighted blue, and then click the Add Part Type to Discounts button (see example below).
