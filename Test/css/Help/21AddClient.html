2.1 Add New Customer


Purpose of this function: To add a customers details to the system to be used when processing orders and generating financial documents.


Navigation to add a new customer


The Add New Customer screen will display where you will be required to input all the required information about the customer.




Screen Controls




The numbers indicated on the screen represent the following controls:

  1. Customer tab: on the click of this tab the required Customer information will be displayed (as shown in the figure above).
  2. Discount tab: on the click of this tab the required Customer discount information will be displayed (as shown in the figure below).
  3. Customer Name textbox: the customer name is entered here.
  4. Account Name textbox: the customer’s account name is entered here.
  5. VAT Number textbox: enter the Customers VAT number here.
  6. Status combo box: select the Customer status from here.
  7. Address textbox: enter the Customer delivery street address here.
  8. City textbox: enter the Customer’s delivery address’s city name here.
  9. Province combo box: select the Customers delivery address’s Province from here.
  10. Zip Code textbox: enter the Customer’s delivery address’s postal code here.
  11. Add Contact button: click here if you wish to add another contact person for the Customer.
  12. Remove Contact button: click here if you wish to remove a contact person for the Customer.
  13. Contact Name textbox: enter the contact person’s name here.
  14. Contact Number textbox: enter the contact person’s contact number here.
  15. Contact Email textbox: enter the contact person’s email address here.
  16. Job Description textbox: enter the contact person’s job description here.
  17. Add New Customer button: this button will only be clicked, once all the required information has been entered, in order to add a new Customer.



The numbers indicated on the screen represent the following controls:

  1. Customer tab: on the click of this tab the required customer information will be displayed
  2. Discount tab: on the click of this tab the required customer discount information will be displayed
  3. Contract Discount Rate numeric up-down: enter the contract discount rate offered to the customer here
  4. Settlement Discount Rate numeric up-down: enter the settlement discount rate offered to the customer here
  5. Add Discount to Part Type button: click this button to search for a part type to select and add a part discount rate for it
  6. Discount % textbox: enter the part discount rate here
  7. ‘Pompie Pump’ button: this button name depends on the part type listed for discount, click this button to remove the part type from the part discounts list






Process to follow :



Input the customer’s relevant details in each textbox, and select the correct options from the drop-down lists. The customer’s company name and physical address must be provided. The customer’s status must be set as ‘Active’ in order to be able to process the orders for the customer, if you set the status as ‘In-Active’ then this means you no longer have provide business services to this customer and so customer orders may not be processed for a customer with and ‘In-Active’ status.



Input the customer’s relevant contact person’s details in each textbox. The contact person’s name and contact details must be provided, as well the job title that they hold in the company. Multiple contact persons can be added under one customer.



Click the Add Contact button to add another contact person’s details under the current customer. The same controls will be displayed below the former contact person’s details.

The Add Customer Screen automatically opens the Customer tab. To set the contract discount rate, settlement discount rate and part discount rates, click on the Discounts tab near the top of the page. Set the contract discount rate and settlement discount rate in the relevant textboxes.



Click on the Add Discount to Part Type button to add a new part discount rate. An Add Part Type message-box will appear for you to search for the part type which you would like to provide a discount for this specific customer. This discount will only be used when this customer orders the selected part type.



Once you’ve clicked the Add Discount to Part Type button, the Add Part Type message-box will display. Input the part type’s search criteria in the textbox, and then select what the criteria should be filtered by in the drop-down list. Click the Filter button to then have the search results displayed in the list box.



Click on one of the parts displayed in the listbox so that it is highlighted blue, and then click the Add Part Type to Discounts button to add the part’s details to the part discounts list and later set the discount.


If the part type you are looking for does not display in the listbox then click the Add New Part Type to System button to then open the screen to add a new part type and then return to this Add Customer screen. Once you have added all the part type’s details on the screen and then returned to this one, you will then need to repeat the search (see above paragraph on how to search for a part type) and can then continue to click on the part type and click the Add Part Type to Discounts button.


Once you are finished searching and adding the part types to provide discounts for, you can click the Close button, this will then close Add Part Type message-box and allow you to set the part type’s discount rate in the relevant textbox in the table.




In the Parts with Discounts table, each field in the Name column is a button. In this case the button is named ‘Pompie Pump’, click the ‘Pompie Pump’ button to remove it from the Part Discount rates.


Click the Customer tab to return to the customer’s details information once you are done setting the discounts.

When you have completed adding the new customer’s details, click the Add New Customer button to save the information.



Once you’ve input all the required details correctly and clicked the Add New Customer button, this message will display confirming that the new customer’s details were saved successfully. Click the OK button to dismiss the notification.


System Error Notifications and Troubleshooting



The system will highlight empty fields that require information to be provided, as well as fields which have the wrong information in. A customer may only be successfully added if all the required information has been provided and the information is in the correct format.



An error message similar to the one above may also be displayed at the bottom left of the screen near the Add Customer button, describing the type of error to explain to you why the customer cannot be added. Read the error message to understand what is wrong and click on the textboxes which have the wrong information causing the error, and correct the information. Once all the incorrect information in the textboxes has been corrected, click on the Add Customer button to then try again to successfully add the customer’s details.



An error message similar to the above example may also display by the incorrect textbox providing details on what is wrong with the input data. Read the error message to understand what is wrong and click on the textbox that it is referring to and correct the information. Once the incorrect information in the textbox has been corrected, click on the Add Customer button to then try again to successfully add the customer’s details.
